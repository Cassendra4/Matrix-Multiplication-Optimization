
Serial code for matrix multiplication

	Source Code: serial.cpp
	
	Compile:  g++ serial.cpp -fopenmp -o serial	
	Run:  
	./serial <matrix_size>


Parallel code for matrix multiplication

	Source Code: parallel.cpp
	
	Compile:  
g++ parallel.cpp -o parallel -fopenmp	
	Run:  
	
./parallel <matrix_size>



Optimized code for matrix multiplication

	Source Code: optimized.cpp
	
	Compile:  g++ optimized.cpp -o3 -o opt3 -fopenmp	
	Run:  
	
./optimized <matrix_size>
